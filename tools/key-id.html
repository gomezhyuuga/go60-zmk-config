Go60 Key Identifier — click here, then press a key on the Go60

Waiting for a keypress…

Maps each KeyboardEvent.code back to its POS_* label on the Go60 matrix. Tap each key to identify it. Home-row mods (A S D F J K L ;) report on tap; holding sends a modifier and may not produce a discrete event here. Layer / magic / bootloader / mouse keys produce no host event and won't be detected — those are easy to identify by elimination.
